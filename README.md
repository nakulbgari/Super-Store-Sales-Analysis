📊 Super Store Sales Dashboard
🎯 Objective

To contribute to the success of a business by utilizing data analysis techniques, specifically focusing on time series analysis, to provide valuable insights that enhance decision-making and drive growth.

🧩 Project Overview

This Power BI project analyzes the Super Store Sales dataset to uncover key insights into business performance, customer behavior, and sales trends.
The dashboard provides an interactive, data-driven view of how different factors such as region, category, and time influence overall profitability.

🧠 Key Learnings

Applied time series analysis to identify seasonal patterns and sales trends over time.

Created interactive dashboards using filters, slicers, and drill-down functionality for dynamic exploration.

Built data relationships and DAX calculations to support deeper insights and performance metrics.

Designed custom visualizations (bar charts, donut charts, line graphs, KPIs) to communicate insights effectively.

📈 Insights Derived

Identified top-performing categories and sub-categories contributing most to revenue.

Detected sales seasonality and forecasted potential sales growth using time trends.

Analyzed regional performance to find areas needing improvement.

Evaluated customer segments to enhance marketing and sales strategies.

🛠️ Tools & Technologies

Power BI Desktop – for dashboard creation and visualization

DAX (Data Analysis Expressions) – for calculations and KPIs

Power Query – for data cleaning and transformation


🚀 Conclusion

This project demonstrates how effective data analysis and visualization can turn raw data into actionable business insights.
By focusing on time series trends, this dashboard supports better forecasting, planning, and strategic business decisions.
